### 5. Config View Redesign (Tab 7)
4 tool-based sections (GitHub, Azure, Jira, System) shown one at a time, full width. Left/right to switch.

### 6. API Token Config
GitHub + Azure tokens configurable in Config view. Auth priority: config → CLI → env.

- **Search**: `useInput` captures raw text, `useRouteShortcuts` disabled; status bar shows `Search: /`
